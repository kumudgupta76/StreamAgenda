# **App Name**: StreamAgenda

## Core Features:

- Task Input: Allow users to input tasks and checklist items for their agenda.
- Live Stream Integration: Allow copy of agenda data that can be added into OBS.
- PPT Export: Generate a PPT presentation of the agenda. (via Markdown)

## Style Guidelines:

- Primary color: Deep Indigo (#663399), chosen to evoke professionalism.
- Background color: Very light grey (#F0F0F0), provides a clean, unobtrusive backdrop.
- Accent color: Teal (#008080), will contrast well with the Deep Indigo and the light grey.
- Body and headline font: 'Inter', a grotesque-style sans-serif, is a good fit for this app and suits both headlines and body text.
- Clean, slide-based layout, making it look and feel like a presentation.
- Subtle transitions between agenda items.
